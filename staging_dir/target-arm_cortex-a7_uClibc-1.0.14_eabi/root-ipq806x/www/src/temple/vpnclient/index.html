After OpenVPN bridging is successful, this router will automatically obtain an IP from the OpenVPN Server and it will become inaccessible. If you want to access this router's UI, you need to connect your device to this router and manually change the device's IP address to this router's origin subnet. For example, the origin gateway of this router is 192.168.8.1, after the bridging is successful, it will obtain an IP such as 10.8.0.2(Server IP is 10.8.0.1), and your PC connected to this router will obtain an IP from the server such as 10.8.0.60. You have to change your PC's IP address to 192.168.8.x, then access 192.168.8.1 to enter this router's UI. Or you can restore the network by holding down the reset button for 4 seconds.
